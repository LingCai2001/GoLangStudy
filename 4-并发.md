## `go`关键字
Go语言支持并发，我们只需要通过`go`关键字来开启`goruoutine`即可
`goroutine`是轻量级线程，`goroutine`的调度是由Golang运行时进行管理的。
```go
// ----------------语法----------------
go 函数名(参数列表)
// ----------------示例----------------
go f(x, y, z)
// ----------------实例----------------
func say(s string){
	for i := 0; i < 5; i ++ {
		time.Sleep(100 * time.Millisecond)
		fmt.Println(s)
    }
}
func main(){
	/* 由两个goroutine交替输出 */
	go say("world")
	say("hello")
}
```
